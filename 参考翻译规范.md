# 参考翻译规范

## 空格
+ 中文、英文和数字之间应加空格

## 标点符号
+ 由于是界面翻译场景，总体保持原有标点符号不变

## 保持不译的场景
+ 计量单位，比如 128 GB
+ 专业术语或缩写，比如 PC，TTO
+ 人名、地点、公司名
+ 配置文件选项的说明，比如 upload_tmp_dir

## 特殊词语
+ 带标点的特殊词语应整体翻译，比如 eg. 翻译为"例如"

### 保持前后一致性
比如, 前面用"同样"，后面用"相同"

### 避免汉语常见低级错误用法
比如, "请稍候" 误作 "请稍后"

### 注意英语词汇中的细微差别
比如 remove 和 delete  
前者应翻译为"移除"（从某个集合中去掉）  
后者应翻译为"删除"（销毁）
